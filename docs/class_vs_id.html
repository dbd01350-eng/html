🎨 CSS 이름표 이야기

클래스(Class)와 아이디(ID)는 어떻게 다를까요?

1. 아이디(ID)는 "나만의 이름" 이에요! 🆔

학교에 '김철수'라는 이름을 가진 친구가 한 명뿐인 것처럼, 아이디(ID)는 한 페이지에 딱 하나만 쓸 수 있어요.

  • 특징: 세상에서 하나뿐인 특별한 부분에 붙여요.
  • 기호: CSS에서는 이름 앞에 # (샵)을 붙여서 불러요.
  • 예시: #my-special-name { ... }

2. 클래스(Class)는 "우리 반 유니폼" 이에요! 👕

우리 반 친구들 모두가 똑같은 '체육복'을 입을 수 있는 것처럼, 클래스(Class)는 여러 명에게 똑같이 입혀줄 수 있어요.

  • 특징: 여러 군데에 똑같은 스타일을 줄 때 사용해요.
  • 기호: CSS에서는 이름 앞에 . (점)을 붙여서 불러요.
  • 예시: .our-uniform { ... }

3. 한눈에 비교해볼까요?

아이디 (ID)

# 기호를 써요!
딱 한 명만!
(나만의 이름표)

클래스 (Class)

. 기호를 써요!
여러 명 다 같이!
(단체 유니폼)

💡 잊지 마세요!
