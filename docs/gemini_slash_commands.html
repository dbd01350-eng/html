Gemini CLI Slash Commands

Essential commands for session management, workspace configuration, and more.

1. General & Help

도움말 및 시스템 일반 정보를 확인합니다.

Command Description
/help (또는 /?)도움말 정보 및 모든 명령어 사용법을 표시합니다.
/aboutCLI 버전 및 시스템 정보를 표시합니다.
/quit (또는 /exit)대화 세션을 종료합니다. (--delete 옵션으로 기록 삭제 가능)
/clear터미널 화면과 스크롤백을 지웁니다.
/docs브라우저에서 Gemini CLI 문서를 엽니다.
/bugCLI 관련 이슈를 GitHub에 등록합니다.
/privacy개인정보 보호 정책을 표시하고 데이터 수집 동의를 관리합니다.
/upgrade사용량 제한 상향을 위한 업그레이드 페이지를 엽니다.

2. Session & Conversation Management

대화 세션과 기록을 관리합니다.

Command Description
/resume (또는 /chat)세션 브라우저를 열거나 체크포인트를 관리합니다. (save, list, resume, share, delete)
/rewind히스토리를 거슬러 올라가 대화나 파일 변경 사항을 되돌립니다.
/restore [id]도구 실행 전 상태로 파일 변경 사항을 복구합니다.
/compress전체 대화 컨텍스트를 요약으로 대체하여 토큰을 절약합니다.
/copy마지막 AI 출력을 클립보드에 복사합니다.

3. Workspace & Context

워크스페이스와 컨텍스트(메모리)를 관리합니다.

Command Description
/memoryGEMINI.md 파일의 계층적 메모리를 관리합니다. (show, refresh, list, add)
/directory (또는 /dir)워크스페이스 디렉토리를 관리합니다. (add, show)
/init현재 프로젝트에 맞춤화된 GEMINI.md 컨텍스트 파일을 생성합니다.
/tools사용 가능한 도구 목록을 표시합니다.
/shells (또는 /bashes)백그라운드 쉘 프로세스 보기를 전환합니다.

4. Extensions & Extensibility

기능 확장 및 에이전트 스킬을 관리합니다.

Command Description
/skills에이전트 스킬(전문 워크플로우)을 관리합니다. (list, reload, enable, disable)
/agents서브에이전트(내장, 로컬, 원격)를 관리합니다.
/mcpModel Context Protocol 서버를 관리합니다.
/extensionsCLI 확장을 관리합니다. (list, install, uninstall 등)
/commands.toml 파일의 커스텀 슬래시 명령어를 관리합니다.
/hooks수명 주기 훅(lifecycle hooks)을 관리합니다.

5. Configuration & Environment

CLI 설정 및 환경을 구성합니다.

Command Description
/settings대화형 설정 편집기를 엽니다.
/themeCLI 시각적 테마를 변경합니다.
/model사용할 모델을 구성합니다. (manage, set)
/vim입력 프롬프트의 Vim 스타일 내비게이션 및 편집을 토글합니다.
/terminal-setupVS Code나 Cursor 같은 에디터를 위한 터미널 키바인딩을 설정합니다.
/ideIDE 통합을 관리합니다. (status, enable, disable, install)
/permissions폴더 신뢰(trust)를 관리합니다.
/plan계획 모드(Plan Mode)로 전환하거나 현재 계획을 복사합니다.
/policies활성 도구 실행 정책 목록을 표시합니다.
/setup-githubAI 기반 이슈/PR 분류를 위한 GitHub Actions를 설정합니다.
/auth인증 방법을 변경합니다.
/editor지원되는 외부 에디터를 선택합니다.
/stats세션, 모델, 도구 사용 통계를 표시합니다.
